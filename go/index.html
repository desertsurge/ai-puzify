黑棋: 0 (吃子: 0)
白棋: 0 (吃子: 0)

围棋基本规则

🎯 游戏目标

通过围地来获得更多领地，同时可以吃掉对方的棋子

⚫ 棋子放置

黑棋先行，轮流在交叉点上放置棋子，不能放在已经有棋子的位置

🔗 气与吃子

棋子相邻的空交叉点称为"气"，当棋子被完全包围（无气）时会被吃掉

🚫 禁止自杀

不能下在会导致自己棋子立即被吃的位置（自杀规则）

🔄 打劫规则

不能立即重复上一个棋盘局面（劫争规则）

🏁 游戏结束

双方连续虚手（放弃落子）后游戏结束，计算领地决定胜负

计分方式： 领地数 + 吃子数 = 总分数
